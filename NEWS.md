# xtpretest 1.0.0

* Initial CRAN release.
* Implements Hsiao (2014) F1/F2/F3 homogeneity tests.
* Implements HC1 robust Wald variants of F2/F3.
* Implements Swamy (1970) parameter heterogeneity test.
* Implements Pesaran (2004) cross-sectional dependence test on FE residuals.
* Produces panel summary statistics with between/within variance decomposition.
* Outputs an automatic estimator recommendation based on sequential test results.
